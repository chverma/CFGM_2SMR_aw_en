Nextcloud & EMail platforms with LDAP

Here we will see how to deploy our private cloud. These services will consist of an LDAP server (OpenLDAP) for the management of our users, an e-mail server with its webmail (RoundCube) and our own platform to host files and be able to make collaborative editions of text/calc documents. (Nextcloud).

First step

Clone this repo and go there.

wget https://github.com/chverma/privateCloud/archive/refs/tags/v1.0.0.tar.gz tar xzvf v1.0.0.tar.gz cd privateCloud-1.0.0/

Install the services

Follow this video tutorial in order to deploy the LDAP, Nextcloud and EMail servicies

Final step

  1. Create five users: admin, boss, human-resources, financial, worker
  2. admin, boss and human-resources can manage all platforms as an administrator. TIP: We've to make a group.
  3. financial and worker have an a user account and are included in the employers group.
